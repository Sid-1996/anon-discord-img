特色功能

拖曳 / 貼上 / 選取

自動暴雷遮罩

上傳後檔案自動加上 SPOILER_ 前綴，Discord 頻道需點擊才能顯示。

局部馬賽克筆刷

在瀏覽器內用 Canvas 塗抹遮蓋敏感區域，圖片不外送，隱私安心。

Twitter / X 貼文

直接貼上推文連結，自動下載圖片以暴雷附件送出，或讓 Discord 原生 embed。

Pixiv 圖片網址

貼上 i.pximg.net 圖片網址，伺服器 proxy 下載後以附件送出。

3 天內可撤回

發送歷史保留 3 天，一鍵撤回 Discord 上的訊息，不怕發錯。

完整部署教學

1

取得 Discord Webhook 網址

你需要在 Discord 頻道建立一個 Webhook，讓工具可以發送訊息到該頻道。

  1. 打開你的 Discord 伺服器
  2. 選取你要發圖的頻道（右鍵或點齒輪 ⚙️）
  3. 進入 整合 → Webhook
  4. 點擊 新增 Webhook，取一個名字
  5. 複製 Webhook 網址（長這樣：https://discord.com/api/webhooks/123456/abc-def-ghi）
你需要頻道的「管理 Webhook」權限。如果是伺服器管理員，預設就有。
2

一鍵部署到 Vercel

點擊下方按鈕，Vercel 會自動將專案 fork 到你的 GitHub 帳號並部署。

  1. 點擊 Deploy 按鈕 (或前往 GitHub 專案)
  2. 登入（或註冊）Vercel 帳號，授權連接 GitHub
  3. 在「Configure Project」頁面填入兩個環境變數：
    1. DISCORD_WEBHOOK_URL — 貼上第一步拿到的 Webhook 網址
    2. ACCESS_TOKEN — 自訂一組密碼（例如 MyR4nd0mK3y!）
  4. 按 Deploy，等待約 1-2 分鐘

部署完成後，你的專屬網址就是 https://你的專案名.vercel.app。

Vercel 免費方案即可使用，無需綁定信用卡。所有程式碼在 GitHub 上完全開源。
3

開始匿名發圖

打開你的 Vercel 網址，即可看到上傳介面。

  1. 在頁面右上角（或設定區塊）找到 Token 欄位
  2. 輸入你在部署時設定的 ACCESS_TOKEN 值，按確認儲存（同一瀏覽器只需一次）
  3. 拖曳圖片到頁面、按 Ctrl+V 貼上，或點選上傳區域
  4. （選用）啟用馬賽克筆刷，塗抹需要遮蓋的區域
  5. 按 確認發送，圖片即以暴雷遮罩 (SPOILER_) 發送到你的 Discord 頻道

也可直接貼上 Twitter/X 貼文連結 或 Pixiv 圖片網址 發送到 Discord。

常見問題

需要註冊 Discord 機器人嗎？
不需要。本工具使用 Discord Webhook 發送訊息，不需要建立或設定 Discord Bot，也不用管理權限或 token。只需在頻道建立 Webhook 即可。
圖片會儲存在伺服器上嗎？
不會。圖片上傳到伺服器後，立即處理並轉發到 Discord，處理完的暫存檔案會立刻刪除。馬賽克筆刷的處理完全在瀏覽器內完成，圖片從未離開你的裝置。
部署到 Vercel 需要付費嗎？
Vercel 免費方案即可使用。免費方案每月有 100 小時執行時間和 100GB 頻寬，對於個人或小社群使用非常足夠。無需綁定信用卡。
ACCESS_TOKEN 忘記了怎麼辦？
在 Vercel 專案設定頁面 (Settings → Environment Variables) 重新設定 ACCESS_TOKEN 的值，然後重新部署即可。
可以發送非圖片檔案嗎？
目前僅支援靜態圖片（JPEG / PNG / WebP）。這是為了維持簡潔的上傳體驗，並確保 Discord 的暴雷遮罩功能正常運作。
這是開源的嗎？可以自己修改嗎？
是的！本專案採用 MIT 授權，完全開源。你可以自由使用、修改、商用。原始碼在 GitHub 上。
