三步驟快速上手

1
一鍵部署到 Vercel
點擊 GitHub 專案 的 Deploy 按鈕， 填入 Discord Webhook 網址與自訂 Token 即可完成部署。
2
取得 Discord Webhook
在 Discord 頻道按齒輪 → 整合 → Webhook → 新增 Webhook， 複製網址並設為環境變數 DISCORD_WEBHOOK_URL。
3
開始匿名發圖
拖曳圖片到下方區域、Ctrl+V 貼上圖片或連結， 按「確認發送」即可在 Discord 以暴雷遮罩顯示。
